# data-structures

学习 JavaScript 数据结构与算法

## 初始化工程

```shell
# 初始化package.json
npm init

# 安装Typescript
npm i typescript ts-node ttypescript @types/node -D

# 初始化tsconfig.json
./node_modules/.bin/tsc --init

# 创建源码
mkdir src

# 添加必要的开发scripts
```

## 本地调试

```shell
npm run debug
```

## 构建

```shell
npm run build
```
